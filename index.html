Foody

Foody is an android app that tackles the problem with walking home late at night. It feels vulnerable for some people when they have night classes and need to head home by themselves, in spots that are not as well circulated as the city center. In this application, users will input in their destination, and search for other people that are walking in a similar destination as them. They will be matched with someone else based on their Facebook mutual friends, location distances, and time. Once matched, users have the option to contact each other through Facebook and go on from there

Uses
- Google Maps Android API
- Google Firebase
- Facebook API

Parker

Me and 3 teammates created an Android app during a 24 hour Hackathon and it allows users to rent out their parking spaces to other users and also rent other users' parking spaces for a certain rate.

Uses
- Google Maps API for users to view available parking
- Google Places API to allow users to search various areas for parking
- Google Firebase to store user data
- Google Sign In to allow users to create accounts and manage their parking spots
